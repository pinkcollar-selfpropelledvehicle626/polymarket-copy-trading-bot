## How To Use Commands

You interact with the program by typing commands into the terminal window. A command consists of a prefix followed by your specific instructions. Press Enter after typing a command to send it to the AI.

### Basic Interaction
Type `gemini` followed by your question. For example: 
`gemini what is the best way to organize files`

The tool will display the answer directly in your terminal. 

### Writing Code
You can use the tool to write computer code. Type `code` then describe the function you need.
`code create a script to rename multiple photos`

The program returns the code block. You can copy this text and save it as a file on your computer.

### Slash Commands
Standard slash commands manage your settings and tool preferences:

*   `/help` – Displays a list of all current functions.
*   `/version` – Shows the current software version installed.
*   `/clear` – Erases the text visible in your window.
*   `/exit` – Closes the terminal program safely.

## Troubleshooting Common Issues

If the software does not respond as expected, check these common items first.

**The terminal window closes immediately.**
This often means your internet connection is blocked by a firewall. Check your Windows Security settings and ensure that the application has permission to access the internet.

**The terminal shows an error about the API key.**
The tool requires an active connection to Google services. If you have an existing Gemini API key, you can enter it by typing `/key` followed by your secret string. If you do not have one yet, the tool will guide you through a free registration process the first time you run it.

**Text appears slowly.**
This depends on your network speed. If the AI is busy, responses may take a few extra seconds to populate. 

**Commands are not working.**
Ensure you are using the correct syntax. Always include the slash symbol for configuration commands. If you are unsure, type `/help` to see the complete list of valid commands for your current version.
